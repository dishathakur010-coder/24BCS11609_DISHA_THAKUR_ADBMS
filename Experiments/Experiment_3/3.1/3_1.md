# SQL COUNT Using CASE - Count Students Scoring Above 80 Marks

## 📌 Problem Statement

Write a SQL query to count the number of students in each department who have scored **more than 80 marks**.

- The count column should be aliased as **`Dept_HighScore_Count`**.
- The `student` table contains the following columns:
  - `St_id`
  - `St_name`
  - `Marks`
  - `Department`

Use the `CASE` statement inside the `COUNT()` function to count only the students satisfying the given condition.

---

## 📝 SQL Query

```sql
SELECT Department,
       COUNT(CASE
                WHEN Marks > 80 THEN 1
                ELSE NULL
             END) AS Dept_HighScore_Count
FROM student
GROUP BY Department;
```

---

## 📖 Explanation

- `CASE WHEN Marks > 80 THEN 1 ELSE NULL END` checks each student's marks.
- If the student has scored **more than 80**, it returns `1`.
- Otherwise, it returns `NULL`.
- `COUNT()` counts only the non-NULL values.
- `GROUP BY Department` groups students by department and returns the count for each department.

---

## ⚠️ Key Points

- `COUNT()` ignores `NULL` values.
- `CASE` allows conditional counting within aggregate functions.
- `GROUP BY` is required to calculate counts department-wise.

---

## 📥 Example

### Student Table

| St_id | St_name | Marks | Department |
|------:|----------|------:|------------|
| 1 | Alice | 92 | Math |
| 2 | Bob | 75 | Math |
| 3 | Charlie | 88 | Physics |
| 4 | David | 95 | Physics |
| 5 | Emma | 70 | English |

### Output

| Department | Dept_HighScore_Count |
|------------|---------------------:|
| English | 0 |
| Math | 1 |
| Physics | 2 |

---

## ⏱️ Time Complexity

- **O(n)**

where:

- `n` = total number of rows in the **student** table.

---

## 💡 Key Concepts

- **CASE** is used to apply conditions within SQL queries.
- **COUNT()** counts only non-NULL values.
- **GROUP BY** creates separate groups for each department.
- This technique is known as **conditional aggregation**.

---

## 🔄 Alternative Solution

The same result can also be achieved using `SUM()`:

```sql
SELECT Department,
       SUM(CASE
              WHEN Marks > 80 THEN 1
              ELSE 0
           END) AS Dept_HighScore_Count
FROM student
GROUP BY Department;
```

Both approaches produce the same result.

---

## ✅ Result

The query returns each department along with the number of students who scored **more than 80 marks**, using `CASE` inside the `COUNT()` function for conditional aggregation.
